```text
# 커머스 도메인 모델 (Domain Model)
(도입 — 도메인 수 현행화)
## 공통 규약
## 공용 값 객체 (Money)
## 1. 회원 (member) (+MemberAddress: 필드·기본배송지 불변식·오퍼레이션)
## 2. 상품 (product) (+Category·ProductImage: 필드·상태·불변식·오퍼레이션)
## 3. 재고 (stock) (+StockRestocked 이벤트 발행 서술)
## 4. 장바구니 (cart)
## 5. 쿠폰 (coupon)
## 6. 주문 (order) (부분취소 누계 필드·라인 취소 전이·ReturnStatus 상태 머신 반영)
## 7. 결제 (payment) (부분환불 필드·전이 반영)
## 8. 위시리스트 (wishlist) (신규 — 재입고 알림 RestockNotification 포함)
## 9. 리뷰 (review) (신규)
## 10. 문의 (inquiry) (신규)
## 크로스 도메인 정책 (체크아웃 오버로드: 전체·선택·바로구매·미리보기 / 부분 취소·라인 반품 조율 /
  리컨실·스윕의 app-batch 소재 / 아웃박스 발행 경로)
## 도메인 이벤트 명세 (현행 이벤트 전체 — OrderPaid 외 사실표가 확인한 것)
## 상태 전이 요약 (신규 상태 머신 추가: ReturnStatus 등)
```
